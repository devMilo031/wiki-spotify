# Microservicios en Spotify

Spotify utiliza una arquitectura basada en microservicios para escalar y mantener sus funcionalidades de forma independiente.

## Ventajas

- Despliegues rápidos y seguros.
- Escalabilidad horizontal.
- Aislamiento de fallos.
## Ejemplos de microservicios

- User Service: manejo de cuentas y perfiles.
- Playlist Service: gestión de listas de reproducción.
- Recommendation Service: algoritmos de sugerencia.
- Streaming Service: control de la entrega de audio.
## Tecnologías empleadas

- Java, Python y C++.
- gRPC y REST para comunicación.
- Docker y Kubernetes para orquestación.
- Google Cloud Platform (GCP) como infraestructura base.
