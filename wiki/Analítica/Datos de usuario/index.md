# Análisis de datos de usuarios

Spotify recopila datos de uso para mejorar la experiencia y personalizar el contenido.

## Tipos de datos recolectados

- Canciones escuchadas.
- Tiempo y frecuencia de reproducción.
- Dispositivos utilizados.
- Interacciones sociales.
## Propósito

- Optimizar las recomendaciones.
- Mejorar el diseño de la app.
- Identificar tendencias globales.
## Privacidad

Spotify cumple con el Reglamento General de Protección de Datos (GDPR) y permite a los usuarios descargar su historial y controlar qué datos comparten.
