# Machine Learning en Spotify

Spotify utiliza algoritmos de aprendizaje automático para personalizar la experiencia musical.

## Aplicaciones principales

- Clasificación de géneros y estados de ánimo.
- Detección de patrones de escucha.
- Generación de playlists automáticas.
## Modelos usados

- Redes neuronales convolucionales (CNN): para analizar audio.
- Modelos colaborativos: para encontrar similitudes entre usuarios.
- NLP (procesamiento del lenguaje natural): para entender títulos y descripciones de canciones.
## Ejemplo real

Spotify usa un modelo de Word2Vec musical que representa canciones como vectores numéricos, lo que permite encontrar temas similares según el gusto del usuario.
